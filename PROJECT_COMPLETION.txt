================================================================================
  USAGE QUICK START
================================================================================

1. CONFIGURE PAYMENT:
   Admin → TRM Counseling → Settings
   - Enable Stripe, PayPal, and/or Offline
   - Add payment credentials
   - Set offline payment instructions

2. CREATE EVENT:
   Admin → TRM Counseling → Events/Webinars → Add New
   - Set event name and details
   - Configure capacity (1 for single, 5+ for multiple)
   - Set session duration and buffer time
   - Configure payment if required
   - Set event date/time or recurring pattern
   - Publish

3. ADD BOOKING FORM:
   Edit a WordPress page
   - Add shortcode: [trm_booking_form event_id="123"]
   - Replace 123 with your event ID
   - Publish

4. USERS CAN NOW:
   - View available slots based on event capacity
   - Book time slots with automatic availability checking
   - Select payment method (if required)
   - Receive confirmation email with details
   - View booking in admin dashboard
